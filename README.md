# 🎵 Beat Portfolio Builder

**Showcase your tracks made with Bakery Beats!**

## Features

✅ **Create Artist Portfolios**
- Add artist name and bio
- Showcase up to 10+ tracks
- Add track details (genre, BPM, streams)

✅ **Share Your Tracks**
- Link to Spotify, YouTube, SoundCloud
- Show streams/views count
- Display release dates

✅ **Beautiful Gallery**
- Professional dark theme
- Responsive mobile design
- Easy browsing

✅ **No Database Needed**
- Uses browser local storage
- All data saved locally
- Works completely offline

## How to Use

1. **Open the tool**: Go to [beat-portfolio-builder.bakerybeats.io](https://bakerybeats.github.io/beat-portfolio-builder/)
2. **Create Portfolio**: Click "Create Portfolio" tab
3. **Fill in details**: Add artist name, bio, tracks
4. **Publish**: Click "Publish Portfolio"
5. **Share**: Your portfolio is now in the gallery!
